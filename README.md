# Shuk

Claude Code plugin marketplace by idl3.

## Plugins

| Plugin | Description |
|--------|-------------|
| [diakon](https://github.com/idl3/diakon) | Multi-project workspace orchestration |

## Install

```bash
/plugin marketplace add idl3/shuk
/plugin install dk@shuk
```
